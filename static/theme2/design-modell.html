Modell Agrammon

Das Simulationsmodell Agrammon steht in drei Varianten zur Verfügung — gleicher Aufbau, gleiche Bedienung, unterschiedlicher Massstab.

01 — Betrieb

Einzelbetriebsmodell

Basiert auf wissenschaftlich gut dokumentierten Modellparametern und Prozessen. Erlaubt die Eingabe der relevanten Betriebsparameter — es werden ausschliesslich Grössen verwendet, die der Landwirtin ohne besonderen Aufwand bekannt sind.

Starten →
02 — Region

Regionalmodell

Eignet sich für die Berechnung regionaler Ammoniakemissionen. Erlaubt die Eingabe regionaler Tierzahlen und der prozentualen Anteile der Produktionstechnik — und rechnet mit demselben Algorithmus wie das Einzelbetriebsmodell.

Starten →
03 — Kanton

Kantonale Anpassungen

⚠ Eingefroren · offline 2028

Für den kantonalen Vollzug, etwa zur Beurteilung von Baugesuchen. Enthält zusätzliche Minderungsmassnahmen und eine Funktion zum Versand von Datensätzen an die zuständige kantonale Fachstelle.

Starten →

Modellbedienung

Web-Applikation

Agrammon ist eine Web-Applikation, die Sie direkt im Browser bedienen — ohne Installation auf dem eigenen Rechner.

Empfohlen: Firefox · Chrome · Edge (aktuelle Version)

Zur Verwendung benötigen Sie ein Benutzerkonto, in dem Ihre Eingabe-Datensätze gespeichert und jederzeit wieder abgerufen werden. Das Konto richten Sie direkt im Login-Fenster ein.
