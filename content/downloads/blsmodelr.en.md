The R package **bLSmodelR** provides functions to set up and run a backward Lagrangian stochastic (bLS) dispersion model with the statistical software R.

## Characteristics

- First-order Lagrangian stochastic model running in backward temporal mode
- Assumes horizontally uniform and vertically varying Gaussian turbulence
- Uses generalized Langevin equations for trajectory calculations
- Calculates concentration-to-emission (C/E) relationships from trajectory ensemble analysis
- Does not model deposition or chemical reactions
