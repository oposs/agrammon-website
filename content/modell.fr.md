## Utilisation du modèle

Le modèle Agrammon est une application web que vous pouvez utiliser directement avec un navigateur, sans installation sur votre ordinateur. Les navigateurs que nous vous recommandons sont Firefox, Google Chrome ou Microsoft Edge (une version actuelle de ces navigateurs).

Pour utiliser cette application, vous devez créer un compte d'utilisateur : vous y mémoriserez les paramètres à introduire et pourrez ensuite y revenir à tout moment. Vous avez la possibilité d'aménager votre compte d'utilisateur directement sur la fenêtre de connexion du modèle Agrammon.
