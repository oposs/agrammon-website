## Modellbedienung

Das Modell Agrammon ist eine Web-Applikation, die Sie direkt mit dem Web-Browser bedienen können, ohne Installation auf dem eigenen Rechner.

Wir empfehlen Ihnen die Browser Firefox, Google Chrome oder Microsoft Edge (jeweils eine aktuelle Version).

Zur Verwendung von Agrammon benötigen Sie ein Benutzerkonto, unter dem die von Ihnen eingegebenen Input-Parameter-Datensätze gespeichert und später jederzeit wieder abgerufen werden können. Ihr Benutzerkonto können Sie direkt auf dem Login-Fenster des Modells Agrammon einrichten.
