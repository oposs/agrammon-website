## Usage of the Model

The model Agrammon is a web application that you can operate directly with your web browser without installing the application on your computer.

We recommend that you use the browser Firefox, Google Chrome or Microsoft Edge (a current version). To use Agrammon you need a user-account, in which the input parameter dataset that you enter is saved and can be called up again at any stage. You can set up your user-account directly on the log-in window of the Agrammon model.
